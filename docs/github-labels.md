# Standard etykiet GitHub — StageSync

Dokument opisuje wspólny standard prefiksów i kolorów etykiet używanych w repozytorium.

## Moduły (`scope:*`)

| Etykieta | Znaczenie | Kolor |
| --- | --- | --- |
| `scope:web` | Frontend / Client Shell / UI | `#1d76db` |
| `scope:desktop` | Tauri / Native OS / Sidecar | `#5319e7` |
| `scope:server` | Host / Node.js / API / Sockets | `#0052cc` |
| `scope:shared` | Pakiety współdzielone / Schematy | `#0e8a16` |
| `scope:timeline` | Silnik osi czasu / Audio / Transport | `#fbca04` |

## Typy (`type:*`)

| Etykieta | Znaczenie | Kolor |
| --- | --- | --- |
| `type:feat` | Nowa funkcja | `#a2eeef` |
| `type:bug` | Poprawka błędu | `#d73a4a` |
| `type:docs` | Zmiany w dokumentacji | `#0075ca` |
| `type:refactor` | Optymalizacja / Czystka kodu | `#cfd3d7` |
| `type:chore` | Konfiguracja CI/CD / Zarządzanie repozytorium | `#7057ff` |

## Statusy (`status:*`)

| Etykieta | Znaczenie | Kolor |
| --- | --- | --- |
| `status:blocked` | Zablokowane przez inne zadanie | `#b60205` |
| `status:needs-review` | Czeka na przegląd | `#fbca04` |
