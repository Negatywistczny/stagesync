# `@stagesync/ui`

Design system StageSync: tokeny CSS (`--ss-*`) i kanoniczny `Button`.

## Użycie

```ts
import { Button } from "@stagesync/ui";
import "@stagesync/ui/tokens.css";
```

## Eksporty

| Ścieżka | Zawartość |
|---------|-----------|
| `@stagesync/ui` | `Button`, typy |
| `@stagesync/ui/tokens.css` | `:root` — kolory, spacing, typografia |
